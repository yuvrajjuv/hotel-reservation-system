# 🏨 Smart Hotel Reservation System

A smart hotel room allocation system that automatically assigns the best available rooms while minimizing travel time between floors and corridors.

This project demonstrates algorithm optimization with a modern React-based UI.

---

## 🚀 Features

- Smart room allocation algorithm
- Minimum travel time calculation
- Real-time room availability
- Random occupancy simulation
- Interactive UI with floor layout
- Modern responsive design

---

## 🧠 How It Works

The algorithm allocates rooms in a way that minimizes the total travel time between selected rooms.

Travel time includes:
- Vertical travel (between floors)
- Horizontal travel (between rooms)

This ensures guests receive rooms that are closest to each other.
